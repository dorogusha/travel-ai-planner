Specification: Side Panel Component

Task Description

Create a Vue component for the current project that:

- Implements a collapsible side panel (drawer) on the left side of the screen.
- Uses Quasar's QDrawer and QTabs components.
- Contains two tabs: "Info" and "Chat".
- The "Info" tab displays location

 details (name, address, current status, history, legends, architecture, notable people, interesting facts).

- The "Chat" tab provides an interface for AI-powered Q&A about the selected location.
- The panel can be toggled open/closed via a button in the top menu or by clicking on a map location.
