# Specification: Map Component

## Task Description

Create a Vue 3 component for current project that:

- Occupies the entire visible area of the screen (fullscreen).
- Uses the Google Map library to display the map.
- On load, determines the user's geolocation and centers the map on these coordinates.
- If the user does not grant geolocation permission, centers the map on default coordinates (Warsaw).
- Contains only the basic map display functionality, without extra logic.
